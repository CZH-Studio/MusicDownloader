# 音乐下载器

轻量级的命令行工具，从QQ音乐、网易云音乐和哔哩哔哩平台下载音乐

## 环境配置

```sh
pip install -r requirements.txt
```

## 使用

```sh
python main.py
```

## 已知的问题
